# 🧠 Cognitive Graph Theory（认知图论）

## 📖 项目简介

这是一个基于能量最小化原则的认知计算框架，用图论描述知识结构、用群论分析认知操作、用动力学模拟思维演化。本项目包含完整的理论模型、严格的代数验证实验，以及从51到111概念的四规模系统性计算实验。核心发现表明：认知能耗最小化是驱动概念压缩与原理迁移自然涌现的第一性原理，系统在不同知识复杂度下展现出与人类认知高度一致的自适应规律。

本工作由一位大二本科生独立完成，核心思想源于对自我思维过程的内省与抽象，后经计算实验验证，其结论与当前具身智能、自由能原理等前沿研究方向形成了深度的原理性共鸣。

本项目的核心贡献在于：

- **整合多学科概念**：融合认知科学、数学物理、计算机科学，为知识表征、学习与创造性思维提供一个统一解释框架。
- **发现新颖认知规律**：通过四规模（51/71/91/111概念）模拟实验，观察到概念压缩密度随知识规模急剧下降、111概念处出现策略相变等与人类学习轨迹高度吻合的现象。
- **代码完整验证**：完全基于基础科学库（numpy/networkx等）实现，无任何黑盒依赖，所有现象均可复现、可追踪。
- **实践落地探索**：另有一个基于本理论的个性化教育原型系统，通过问卷获取认知参数，驱动认知网络演化，为学习者提供自适应建议（见『基于认知的个性化教育实验』仓库）。

## 核心思想与方法

### 认知图论的两大公设

1. **认知时空（几何描述）**  
   认知状态可表示为动态图 $G(V,E,t)$，其中节点 $V$ 为概念，边 $E$ 为概念间的关联，边权重为认知能耗 $E_{ij}(t)$。网络的几何特性（度分布、聚类系数、路径长度、模块性）刻画了知识结构。

2. **能量动力学（驱动机制）**  
   系统演化受能耗最小化驱动，认知能耗函数定义为：
   $$E_{ij}(t) = \alpha \cdot (1 - \text{sim}_{ij}) + \beta \cdot \frac{1}{1 + H_{ij}(t)} + \gamma \cdot \text{div}_{ij}$$
   其中 $\text{sim}_{ij}$ 为语义相似度，$H_{ij}(t)$ 为历史熟练度（模拟遗忘与学习），$\text{div}_{ij}$ 为结构差异性。系统通过反复“遍历”（认知操作）降低全局认知自由能 $F(G)$，从而自组织演化。

### 涌现现象

从上述两个简单公设出发，系统自发产生了两种与人类认知高度一致的宏观行为：

- **概念压缩**：当一组概念频繁协同激活且对外连接能耗较高时，系统将其压缩为一个宏观节点，显著降低全局能耗。例如，“算法+神经网络+机器学习”最终涌现为“强化学习”这一高阶概念。
- **第一性原理迁移**：通过高普适性原理节点（如“模式识别”、“优化”）建立低能耗捷径，实现跨领域思维跳跃。例如，“发展心理学→模式识别→元认知”的迁移路径模拟了类比推理的创造性过程。

### 代数结构（理论深化）

为给认知操作提供严格的数学基础，我们引入群论结构：
- 基本认知操作（遍历、学习、遗忘、压缩、迁移）构成**半群**，满足封闭性与结合律。
- 认知对称群 $\mathcal{G}$ 刻画了网络的不变性，Noether型命题指出连续对称性对应守恒量（如能量、结构熵）。
- 连续时间演化可用**李群方程** $\frac{dG(t)}{dt}=A(t)G(t)$ 描述，为分析认知过程的平滑性提供了理论工具。

代数验证实验（5组）证实了上述结构的计算可行性，详见论文第5节及 `src/algebra/` 模块。但严格的数学推导仍需进一步展开。

## 主要发现与实验结果

我们在 51、71、91、111 概念网络上进行了系统性对比实验，涵盖物理学、数学、计算机科学、认知科学等跨领域知识，每个规模均运行10,000次迭代。实验对比了四种范式：随机网络基线、简单Q-learning、预设算法模型（模拟传统认知计算）、自然涌现模型（本文模型）。

### 客观指标与定量验证
为增强模型的可解释性与可复现性，我们设计了一套客观的定量指标来刻画系统宏观行为与参数敏感性：

* **压缩阈值扫描（threshold scan）**：扫描压缩协同性阈值（0.5–0.9），记录不同阈值下的压缩事件数量，确保所观察现象不是由单一阈值设定造成的。（见 `src/analysis/threshold_scan.py`，结果保存在 `results/analysis/add_scan/`。）
* **压缩势（compression potential）**：计算每次压缩事件内部平均能耗与外部平均能耗之比 Φ，分析其分布和规模依赖性。（见 `src/analysis/run_potential_analysis.py` / `src/analysis/run_potential_analysis_en.py`，结果保存在 `results/analysis/potential_analysis/`。）
* **能耗下降速率拟合**：对能耗随迭代的演化曲线进行幂律/指数拟合，提取衰减指数和拟合优度，量化能耗优化效率。（由 `src/analysis/__init__.py` 中的相关函数支持，结果保存在 `results/analysis/objective_metrics/`。）
* **Zipf 分布检验**：检验压缩事件中概念出现频率是否符合 Zipf/幂律分布，类似自然语言词频规律。（同样由分析模块生成。）

### 能耗优化性能

| 概念规模 | 自然涌现模型 | 预设算法模型 | 相对优势 |
|----------|--------------|--------------|----------|
| 51概念   | **86.9%**    | 22.8%        | +64.1%   |
| 71概念   | **72.3%**    | 15.5%        | +56.8%   |
| 91概念   | **61.4%**    | 12.2%        | +49.2%   |
| 111概念  | **47.9%**    | 7.6%         | +40.3%   |

- **自然涌现模型**在所有规模下均显著优于其他范式，且规模衰减幅度最小（45.6%），展现了强大的鲁棒性。
- **预设算法模型**性能随规模急剧下降（降幅67.1%），暴露了预设规则在复杂认知环境中的局限性。

### 概念压缩的规模自适应

| 规模 | 压缩总次数 | 每概念压缩数 | 涌现强度范围 |
|------|------------|--------------|--------------|
| 51   | 466        | 9.14         | 0.79–0.84    |
| 71   | 332        | 4.68         | 0.78–0.84    |
| 91   | 301        | 3.31         | 0.81–0.84    |
| 111  | 43         | 0.39         | 0.78–0.97    |

- **压缩密度急剧下降**（9.14→0.39），完美模拟了人类从初学者广泛建立连接到专家深度精准整合的认知发展轨迹。
- **111概念出现临界现象**：压缩次数骤降85.7%，系统策略从探索性整合转向保守性维持，体现了认知负荷的智能管理。

### 原理迁移的创造性模拟

- **迁移次数**：51概念0次，71/91概念各3次，111概念仅1次。
- **迁移效率**：平均0.35–0.45，最高达0.449（91概念）。
- **枢纽节点**：“模式识别”在7次迁移中出现6次，证明抽象原理是跨领域思维的桥梁。
- **迁移路径稳定性**：相同路径在不同规模下重复出现（如“发展心理学→模式识别→元认知”），验证了其有效性。

这些现象与Wallas创造性思维四阶段理论（准备期、酝酿期、豁朗期、验证期）高度吻合，为创造性思维的计算建模提供了新视角。

### 代数验证实验
- **认知操作半群**：验证了13种基本认知操作（遍历/学习/遗忘/压缩/迁移）在复合下满足结合律。
- **Noether型命题**：在三个不同结构的网络上检测到全局能量、结构熵等守恒量趋势，验证了对称性对应守恒量的思想。
- **轨道-稳定子定理**：确认了在认知状态空间中轨道大小与稳定子大小的关系（尽管网络高度个性化导致同构数为0，但定理在平凡情况下成立）。
- **李群演化演示**：展示了连续时间认知演化的可行性，不同生成元组合导致不同能耗优化效果（能量优化主导策略降低37.8%能耗）。

## 应用探索：个性化教育原型（示例）

基于认知图论，我们开发了一个简单的教育系统原型，展示理论落地潜力：

1. **用户问卷**：通过主观题与选择题来采集学习者的抽象能力、记忆能力等认知参数。
2. **认知图构建**：根据参数初始化个性化认知网络（节点权重、边能耗）。
3. **模拟学习过程**：运行能量最小化演化，观察概念压缩与迁移的发生。
4. **学习路径生成**：根据演化结果，推荐适合的学习路径（如“您可尝试通过‘模式识别’理解新概念‘元认知’”）。

> 当前建议层为接口设计，可接入外部大模型生成自然语言解释，系统仅提供结构化推荐。代码见『基于认知的个性化教育实验』仓库。

## 快速开始

### 环境要求
- Python 3.9+
- 仅需基础库（`numpy`, `pandas`, `matplotlib`, `networkx`, `scipy`, `openpyxl`, `jieba`），无需深度学习框架，保证轻量可复现。

### 安装依赖
```bash
pip install -r requirements.txt
```

### 运行实验
```bash
# 一键运行所有规模对比实验（51/71/91/111概念）
python run_experiments.py

# 单独运行某规模涌现研究
python src/experiments/emergence_study_fixed.py --size 111

# 运行代数验证实验
python src/algebra/algebra_experiments.py

# 运行客观指标分析（阈值扫描、压缩势分析）
python src/analysis/threshold_scan.py
python src/analysis/run_potential_analysis.py   # 中文版本
python src/analysis/run_potential_analysis_en.py # 英文版本
```

### 查看结果
实验结果保存在 `results/` 目录下：
- `51_concepts.xlsx` 等文件：各规模压缩/迁移详细记录。
- `batch_experiments/`：批量实验配置、汇总图表（性能对比图、规模效应曲线）。
- `analysis/add_scan/`：阈值扫描结果。
- `analysis/potential_analysis/`：压缩势分布图与统计摘要。
- `emergence/`：涌现检测详细记录。
- `visualizations/`：认知网络演化图、状态分布图等。

论文核心图表（`performance_comparison.png`, `scale_effect.png`）位于 `paper/` 目录，可直接用于学术展示。
